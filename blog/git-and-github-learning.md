  ðŸ”¹Basic Git Workflow

  Create or modify files

  Add files to staging area

  Commit changes with a message

  Push changes to GitHub

  ðŸ”¹Common Git Commands
  
Check Git version
git --version

Initialize a repository
git init

Check file status
git status

Add files to staging area
git add .

Commit changes
git commit -m "Initial commit"

Connect local repo to GitHub
git remote add origin <repository-url>

Push code to GitHub
git push -u origin main

ðŸ”¹What is a Repository?

A repository (repo) is a folder that contains:

Project files

Git history

Configuration files

Repositories can be local (on your computer) or remote (on GitHub).

ðŸ”¹What is README.md?

README.md is the main documentation file of a repository.

It usually contains:

Description

How to use the project

Technologies used

GitHub automatically displays this file on the repository home page.
